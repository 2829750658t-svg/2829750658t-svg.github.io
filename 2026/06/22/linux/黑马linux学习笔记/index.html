4.9 端口

1.概念

虚拟端口：锁定计算机上特定程序（qq，wx…）

1.1 linux端口（65535个）：

1.公认端口:1-1023，用于系统内置或知名程序的预留使用

如ssh服务的22端口

https的433端口

2.注册端口：1024-49151，松散绑定一些程序和服务

通常可以随意使用

3.动态端口：49152-65535，通常不会固定绑定程序，而是程序对外进行网络链接时，用于临时使用

1.2 查看端口占用namp

语法：namp ip

安装：yum -y install nmap

2.netstat查看指定端口占用

语法：netstat -anp | grep 端口号

安装:yum -y install net-tools

例子：

netstat -anp | grep 6000

tips:

  1. 本机地址0.0.0.0

4.10 进程管理

1.概念

程序运行时，会被操作系统注册为系统中的一个进程

2.查看进程

语法：ps [-e -f]

选项：-e 全部进程

选项：-f 展示全部信息

例子：列出所有进程的全部信息

UID 进程所属的用户id

IPD 进程的进程号

PPID 进程的父id（启动此进程的其他进程）

C 进程cpu占用率(百分比)

STIME 进程启动时间

TTY 启动此进程的终端序号，如显示?则被表示非终端启动

TIME 进程占用cpu时间

CMD 启动命令

3.关闭进程

语法：kill [-9] 进程ID

选项：-9 表示强制关闭此进程；不适用则会向进程发送信号要求其关闭，但是是否关闭看进程自身的处理机制

但是是否关闭要看进程自身的处理机制

4.14 压缩或解压

1.linux常用压缩格式：tar，gzip

2.tar命令（针对.tar和.gz）

.tar:tarball归档，简单封装文件，没有体积减少

.gz:常见为tar.gz，gzip格式压缩文件，有体积减少

语法：tar [-c -v -x -f -z -C] 参数1 参数2… 参数n

选项：

-c 创建压缩文件

-v 显示进度条

-x 解压模式

-f 要加压货压缩的文件，此选项必须位于最后一个

-z gzip模式，不用就是简单的tarball(一般放在选项第一个)

-C 解压目的地
